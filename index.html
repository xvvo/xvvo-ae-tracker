After Effects CEP Extension

AE Time Tracker

Automatic, per-project time tracking that lives inside After Effects. No setup. Just open a project and it starts.

Features

Automatic tracking

Starts the moment you open a project in After Effects, stops when you switch away or close it. Completely hands-free.

Per-project memory

Every project gets its own total. Switch between projects freely — each one accumulates independently.

Session history

Every work session is logged with its date, start time, end time, and duration. Click any project to expand it.

Export reports

Generate a detailed stats report (.txt) or a full CSV of all sessions — exportable to any folder you choose.

Idle detection

If you haven't done anything in AE for 5 minutes, the timer pauses automatically. Resume working and it picks back up.

Offline — no account

Everything is stored locally in your Documents folder. No internet connection, no account, no subscription needed.

How it works

Fully automatic tracking

The extension polls After Effects every 2 seconds to detect which project file is active. You never need to press anything.

01

Open a project in After Effects

The extension detects the project file path automatically. The timer starts and the status dot turns green.

02

Work normally

The timer runs in the background. The extension watches for any AE activity (layer changes, timeline moves, etc.) to detect if you're actively working.

03

Switch apps or close AE

If you switch to another application, Grace Period kicks in — the timer keeps running for 60 seconds before pausing, in case you just alt-tabbed briefly.

04

Come back to AE

Focus returns to After Effects, the timer resumes immediately. The session continues from where it left off.

05

Data is auto-saved

Every 10 seconds, accumulated time is written to disk at Documents/AETimeTracker/data.json. Nothing is lost even if AE crashes.

Settings

Toggles explained

Both settings are saved to disk automatically and persist between sessions.

Grace Period

When After Effects loses focus (you switch to Figma, a browser, etc.), the timer doesn't stop immediately. It waits 60 seconds before pausing — so quick alt-tabs don't break your session. If you return to AE within that window, tracking continues seamlessly and the grace period cancels.

Turn this off if you want the timer to stop the instant AE loses focus.

Idle Pause 5 min

Every 30 seconds, the extension checks whether anything changed in your AE project (layer count, timeline position, selected layers, active comp, etc.). If nothing has changed for 5 minutes, tracking pauses and the dot turns yellow.

As soon as you do anything in AE, it detects the change and resumes automatically. Turn this off if you want the timer to keep running even when you're thinking or stepping away.

Data storage

Where your data lives

All tracking data is saved locally in a JSON file in your Documents folder. No cloud, no account — it's just a file on your computer.

📂 Documents / AETimeTracker / data.json

The file is written every 10 seconds while tracking, so data is preserved even if After Effects crashes. You can open data.json in any text editor to inspect or back up your records manually.

Settings (Grace Period and Idle Pause toggles) are also saved inside this file under a _settings key, so your preferences survive reinstalling the extension.

FAQ

Common questions

The timer isn't starting when I open a project.
Make sure the panel is visible (Window → AE Time Tracker). If it's open but hidden behind another panel, it still works — but check the status dot color. If it's red and says "No project open", try closing and re-opening your .aep file. The extension polls for a saved project path, so unsaved new projects won't register.
It says "Paused" even though I'm actively working.
This usually means After Effects doesn't currently have focus in the OS — another window is on top. Click anywhere inside AE to bring it to the front. If Grace Period is enabled, tracking resumes immediately. If it's disabled, you need to click back into AE.
The Idle Pause keeps pausing me when I'm actually working.
Idle detection works by watching for changes in the AE state hash (layer count, timeline position, selected layers, active comp). If you're doing something like rendering or just reviewing your comp without making changes, it may not detect activity. You can turn off Idle Pause in the settings row at the top of the panel.
My old project data disappeared after reinstalling.
Data is stored at Documents/AETimeTracker/data.json. Reinstalling the extension doesn't touch that file, so your data should be safe. If it's gone, check if Documents is syncing to the cloud (iCloud, OneDrive) and may have been moved.
Can I track multiple projects at once?
No — only one project can be "active" in After Effects at a time. Each project tracks independently, and switching projects stops tracking on the previous one and starts it on the new one.
Does this work on macOS and Windows?
Yes. The extension is a standard CEP panel and runs on both platforms. File paths are handled correctly on both systems.
What After Effects versions are supported?
AE CC 2019 (version 16.0) and later. The manifest targets CSXS 7.0 which covers all modern versions.
How do I reset a project's time?
Open Documents/AETimeTracker/data.json in a text editor, find the entry for your project's file path, and delete or zero out the total field and clear the sessions array. Save the file, then reopen After Effects.
