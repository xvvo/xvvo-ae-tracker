Features

Automatic time tracking

Starts when you open a project, stops when you switch or close. No setup, no pressing anything.

Per-project memory

Session history

Every work session logged with date, start/end time, and duration. Click a project to expand it.

Idle detection

No activity in AE for 5 min? Timer pauses. Work again and it picks back up automatically.

Project stats export

CSV export

Fully offline

How it works

Automatic, always

VAUX polls After Effects every second from inside AE's own engine — not the panel UI. That means tracking continues even if the panel is closed or undocked.

01

Open a saved project

VAUX detects the file path instantly. Status dot turns green, timer starts.

02

Work normally

Timer runs in the background. AE activity is watched every 30 seconds for idle detection.

03

Switch apps or close the panel

Grace Period keeps the clock running for 60 seconds. Closing the panel entirely doesn't stop tracking — it keeps going inside AE.

04

Come back to AE

Focus returns, timer resumes immediately. No lost time.

05

Auto-saved every 10 seconds

Data is flushed to disk regularly. Even if AE crashes, you lose at most 10 seconds.

Settings

The two toggles

Both settings are saved automatically and persist between sessions.

Grace Period

When AE loses focus, the timer keeps running for 60 seconds before stopping — so quick alt-tabs don't break your session. Turn this off if you want the timer to stop the instant AE loses focus.

Idle Pause 5 min

If nothing changes in your AE project for 5 minutes, tracking pauses automatically. Any activity (moving the timeline, changing a layer, etc.) resumes it. Turn off to keep the timer running no matter what.

FAQ

Common questions

The timer isn't starting when I open a project.
Make sure the panel is open at least once (Window → VAUX). VAUX only detects saved .aep files — unsaved new projects have no file path and won't register. Save your project first, then it will start tracking immediately.
It says "Paused" even though I'm actively working.
Another window is likely sitting on top of After Effects. Click anywhere inside AE to bring it to the front. If Grace Period is on, tracking resumes instantly. If it's off, you need to bring AE into focus first.
Idle Pause keeps triggering when I'm active.
Idle detection watches for changes in AE's state (layer count, timeline position, selected layers, active comp). If you're reviewing your comp or doing something AE can't detect as a change, it may think you're idle. Toggle Idle Pause off in the panel to disable this entirely.
Does closing or undocking the panel stop tracking?
No. Tracking runs inside After Effects' own scripting engine, not inside the panel's browser. You can close the panel, undock it, hide it — the timer keeps accumulating. Reopen the panel anytime to see your current totals.
Will I lose data if After Effects crashes?
You'll lose at most 10 seconds. VAUX flushes accumulated time to disk every 10 seconds, so crashes only cost you the current unsaved window. All previous sessions and totals are safe.
How do I reset a project's tracked time?
Your data lives in a file at Documents/AETimeTracker/data.json. Open it in any text editor (Notepad, TextEdit, VS Code), find the entry matching your project's file path, set total to 0, and clear the sessions array to []. Save the file and reopen the panel.
My data disappeared after reinstalling VAUX.
Reinstalling the extension doesn't touch Documents/AETimeTracker/data.json — that file is completely separate from the extension folder. If it's gone, check if your Documents folder syncs to iCloud or OneDrive, which may have moved or conflicted the file.
Does this work on Mac and Windows?
Yes. VAUX is a standard CEP panel and handles file paths correctly on both platforms. It targets After Effects CC 2019 (v16.0) and all versions after that.
